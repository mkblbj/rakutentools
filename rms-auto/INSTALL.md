# インストール手順

## Chrome / Edge / Brave

1. 拡張機能をビルド:
   ```bash
   pnpm build
   ```

2. ブラウザの拡張機能ページを開く:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Brave: `brave://extensions`

3. 右上の「デベロッパーモード」を有効にする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. `build/chrome-mv3-prod` フォルダを選択

## Firefox

1. Firefox用にビルド:
   ```bash
   pnpm build:firefox
   ```

2. Firefox を開いて `about:debugging#/runtime/this-firefox` にアクセス

3. 「一時的なアドオンを読み込む」をクリック

4. `build/firefox-mv2-prod/manifest.json` ファイルを選択

## 初回設定

1. 拡張機能アイコンを右クリック → 「オプション」を選択

2. PINコードを設定（変更時にも必要になります）

3. 店舗情報を入力:
   - 店舗名
   - R-Login ID
   - R-Login パスワード
   - 楽天会員ユーザID
   - 楽天会員パスワード

4. 「保存」ボタンをクリック

## 使用方法

1. 拡張機能アイコンをクリック

2. ログインしたい店舗を選択

3. 自動的にRMSログインページに移動し、認証が開始されます

## トラブルシューティング

### 拡張機能が動作しない

- ブラウザを再起動してみてください
- 拡張機能を一度削除して再インストールしてください

### ログインに失敗する

- オプションページで入力した情報が正しいか確認してください
- パスワードが変更されている可能性があります

### Firefoxで一時的なアドオンが消える

- Firefoxでは開発用アドオンは再起動すると削除されます
- 毎回読み込み直すか、署名付きアドオンとしてパッケージ化する必要があります
