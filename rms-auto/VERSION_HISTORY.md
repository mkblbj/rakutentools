# バージョン履歴

## v0.1.1 (2025-12-14)

### 新機能
✨ データのエクスポート/インポート機能を追加
- JSON 形式でのデータ保存
- 簡単なバックアップとリストア
- 複数デバイス間でのデータ同期が可能

### 初期機能
- Plasmo Framework への移行完了
- Chrome (Manifest V3) 対応
- Firefox (Manifest V2) 対応
- 複数店舗（最大20店舗）のログイン情報管理
- R-Login 自動認証
- 楽天会員自動認証
- RMS利用規約の自動同意
- PIN コードによる設定保護

## 旧バージョンからの移行

jQuery ベースの旧バージョンから以下の改善を実施:

### 技術スタック
- jQuery → React 18
- JavaScript → TypeScript
- 手動ビルド → Plasmo Framework

### 利点
- 型安全性の向上
- 開発時のホットリロード
- 複数ブラウザ対応の簡素化
- モダンな開発体験

### データ互換性
ストレージ形式は互換性を保持しているため、設定の移行は不要です。
